# 안저(Fundus) 경로 설정 가이드

## 현재 상태

`config_real.json`의 안저 경로가 아래와 같이 설정되어 있습니다:

```json
"안저": {
  "folders": [
    "\\\\AFC-210-PC\\Fundus2",
    "\\\\192.168.0.213\\[TODO: 실제 하위폴더명 입력 필요]"
  ],
  "pattern": "\\s(\\d+)\\s",
  "cell": {"row": 13, "col": 3},
  "use_creation_time": true,
  "comment": "find_fundus_path.py 실행하여 192.168.0.213 하위 정확한 경로 확인 필요"
}
```

## 경로 찾기 방법

### 방법 1: 자동 탐색 스크립트 사용 (권장)

```cmd
python find_fundus_path.py
```

이 스크립트는:
- `\\192.168.0.213\` 경로를 자동으로 탐색
- Fundus 관련 폴더를 검색
- 발견된 경로와 하위 구조를 표시

### 방법 2: Windows 탐색기에서 수동 확인

1. Windows 탐색기 열기
2. 주소창에 `\\192.168.0.213\` 입력
3. Fundus 관련 폴더 찾기 (예: `Fundus`, `Fundus2`, `AFC` 등)
4. 전체 경로 복사

## 경로 업데이트 방법

### 1단계: 정확한 경로 확인

예를 들어, 실제 경로가 `\\192.168.0.213\AFC-Fundus\Data` 라면:

### 2단계: config_real.json 수정

```json
"안저": {
  "folders": [
    "\\\\AFC-210-PC\\Fundus2",
    "\\\\192.168.0.213\\AFC-Fundus\\Data"  ← 여기 업데이트
  ],
  ...
}
```

**중요**: JSON 파일에서는 백슬래시(`\`)를 두 번 사용해야 합니다 (`\\`)

### 3단계: 테스트

```cmd
python daily_report_fast.py
```

로그에서 안저 경로가 제대로 인식되는지 확인:
```
📋 안저 (특별항목)
   📂 폴더 1: \\AFC-210-PC\Fundus2
   📂 폴더 2: \\192.168.0.213\AFC-Fundus\Data
   ...
```

## 일반적인 경로 예시

가능성이 높은 경로들:
- `\\192.168.0.213\Fundus`
- `\\192.168.0.213\Fundus2`
- `\\192.168.0.213\AFC-210\Fundus`
- `\\192.168.0.213\share\Fundus`
- `\\192.168.0.213\data\Fundus`

## 문제 해결

### "경로 접근 불가" 오류
- 네트워크 연결 확인
- `\\192.168.0.213\`에 접근 권한이 있는지 확인
- 방화벽 설정 확인

### "폴더를 찾을 수 없음" 오류
- Windows 탐색기에서 수동으로 경로 확인
- IT 담당자에게 정확한 공유 폴더 경로 문의
